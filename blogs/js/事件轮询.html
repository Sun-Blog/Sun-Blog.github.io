# 栈、队列

栈：先进后出

队列：先进先出

# 任务队列

宏任务(macrotask queue)、微任务(microtask queue)

宏任务：Script整体代码、setTimeout、setInterval()、I/O、......

微任务：Promise.then()、......

注意

宏任务开始，执行一个宏任务，询问一次微任务队列，微任务队列有任务，就清空微任务队列，循环往复；宏任务开始，宏任务结束
