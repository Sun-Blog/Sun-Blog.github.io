## 三大框架路由

> Angular  --  ngRouter
>
> React  --  ReactRouter
>
> Vue  --  vue-router

## 安装

> npm install vue-router --save

## 使用

> 第一步：导入路由对象，并调用 Vue.use(VueRouter)
>
> 第二步：创建路由实例，并且传入路由映射配置
>
> 第三步：在Vue实例中挂载创建的路由实例
